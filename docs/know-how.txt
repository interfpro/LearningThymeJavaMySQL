- RESTi tunnused:
    * Klassil on annotation @RestController.
    @RestController is annotation, which marks the class as a controller where every method returns a domain object
    instead of a view (MVC model). It’s shorthand for @Controller and @ResponseBody rolled together.
    * Kindlasti peavad olema objektide mudelid (nt. Greeting.java)
    * RequestMapping meetodi poolt tagastatud objektist luuakse automaatselt JSON objekt, mis siis response bodys täidetakse
    ja cliend'ile tagastatakse.
    The Greeting object must be converted to JSON. Thanks to Spring’s HTTP message converter support, you don’t need to
    do this conversion manually. Because Jackson 2 is on the classpath, Spring’s MappingJackson2HttpMessageConverter is
    automatically chosen to convert the Greeting instance to JSON.
    * Lisaks:
    A key difference between a traditional MVC controller and the RESTful web service controller above is the way that the
    HTTP response body is created. Rather than relying on a view technology to perform server-side rendering of the greeting
    data to HTML, this RESTful web service controller simply populates and returns a Greeting object. The object data will be
    written directly to the HTTP response as JSON.
